│                                             [🔔] [─] [✕]    │  ← 极简顶栏
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                                                             │
│                      主内容区域                              │

| 导航项 | 页面 | 功能 |
|--------|------|------|
| 🏠 启动 | LaunchView | 快速启动、实例列表 |
| ⬇ 下载 | DownloadView | 版本/模组/整合包/资源包/光影 |
| 🎨 创作 | CreatorView | 整合包制作、可视化编辑器 |
| ⚙ 更多 | SettingsView | 启动/下载/主题/账号/关于 |

## 六、项目结构

```
AuroraLauncher/
├── core/                    # 核心基础模块 (Java 8)
├── network/                 # 网络模块 (P2P)
├── ui/                      # 界面层 (Java 17)
│   └── src/main/resources/
│       ├── css/             # 样式文件
│       ├── fxml/            # 布局文件
│       └── i18n/            # 国际化
├── docs/                    # 文档
│   ├── design/              # 设计文档
│   └── superpowers/specs/   # 规范文档
└── modpack/                 # 整合包管理
```
