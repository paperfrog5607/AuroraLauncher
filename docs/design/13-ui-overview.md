# 模块十三：ui - 界面模块总览

## 1. 模块概述

JavaFX界面层，采用PCL风格布局（顶部标签导航+卡片式设计），MVC + FXML架构，CSS自定义主题，中英双语国际化。

## 2. 技术选型

- **框架**: JavaFX 17+
- **布局**: FXML
- **架构**: MVC
- **样式**: CSS自定义主题（PCL风格扁平化）
- **国际化**: ResourceBundle
- **构建**: Gradle

## 3. 设计风格 (PCL风格)

| 特点 | 说明 |
|------|------|
| **顶部标签导航** | [启动] [下载] [更多] 三大主标签 |
| **卡片式布局** | 内容以Card形式组织，支持折叠/展开 |
| **二级子标签** | 下载页/更多页有子标签切换 |
| **快速启动** | 主页顶部直接启动游戏 |
| **扁平化设计** | 无侧边栏，简洁现代 |
| **流畅动画** | 卡片展开/折叠动画效果 |

## 4. 子包结构

com.aurora.ui/

## 6. 页面路由

| 主标签 | 子标签 | 页面 | 说明 |
|--------|--------|------|------|
| 启动 | - | LaunchPage | 主页/快速启动 |
| 下载 | 游戏版本 | DownloadVersionPage | MC版本下载 |
| 下载 | 模组 | DownloadModPage | 模组搜索下载 |
| 下载 | 整合包 | DownloadModpackPage | 整合包下载 |
| 下载 | 资源包 | DownloadResourcePage | 资源包下载 |
| 下载 | 光影 | DownloadShaderPage | 光影包下载 |
| 更多 | 启动设置 | SettingsLaunchPage | Java/内存/参数 |
| 更多 | 下载设置 | SettingsDownloadPage | 并发/代理/镜像 |
| 更多 | 个性化 | SettingsThemePage | 主题/背景/字体 |
| 更多 | 账号 | SettingsAccountPage | 账号管理 |
| 更多 | 关于 | SettingsAboutPage | 版本/更新/协议 |
